Diary Tutorial

This is a simple Diary application demonstrating use of the Dolphin framework for creating applications. It's fairly rudimentary at the moment but it should give some pointers about how to proceed if you want to create your own applications before the ViewComposer becomes fully operational.

Load Diary.pac using the package browser.

You will gain four new classes:

Domain classes

Diary
DiaryEvent

Tool classes

DiaryTool
DiaryEventEditor

Getting started

To bring up a diary try:

MyDiary := Diary new show

The user interface is fairly simplistic but straightforward.You can create additional views onto the same diary using:

(DiaryTool on: MyDiary) inShell

Each view will keep in step.

To shut down the diary you must free it to stop the monitor process from keeping a reference to it. Close all tool windows and evaluate:

MyDiary free.
Smalltalk removeKey: #MyDiary.

To better understand the application framework please visit the Dolphin Education Centre and check out: ApplicationFramework.htm
